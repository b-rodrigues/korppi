# Comments & Reviews

Korppi includes a powerful commenting system for reviews and collaboration.

---

## Adding Comments

### Method 1: Context Menu

1. Select the text you want to comment on
2. Right-click
3. Click **Add Comment**
4. Type your comment
5. Press Enter or click outside to save

### Method 2: Keyboard Shortcut

1. Select text
2. Press a keyboard shortcut (if configured)
3. Type your comment

---

## Comment Highlights

When you add a comment:

- The text is highlighted in your author color
- Hover over highlighted text to see the comment
- Click the highlight to expand the full comment thread

---

## The Comments Panel

The right sidebar contains the **Comments** section:

- Shows all comments in the document
- Sorted by position in document
- Each comment shows:
  - Author name and color
  - Comment text
  - Date/time
  - Status (unresolved/resolved/deleted)

---

## Comment Status

### Unresolved

New comments start as **unresolved**. They appear in the pending count in the status bar.

### Resolved

Mark a comment as resolved when it's been addressed:

1. Click the comment
2. Click **Resolve**

Resolved comments are grayed out but still visible.

### Deleted

Remove a comment:

1. Click the comment
2. Click **Delete**

Deleted comments can be **restored** if needed.

---

## Author Colors

Each author has a unique color:

- Set your color in Profile settings
- Colors make it easy to identify who wrote what
- Comment highlights use the author's color

---

## Filtering Comments

In the comments panel:

- **All** - Show all comments
- **Unresolved** - Only show open comments
- **Resolved** - Only show addressed comments

---

## Comment Threads

Comments support threading for discussions:

1. Click an existing comment
2. Add a reply
3. The thread expands below the original

---

## Best Practices

### For Writers

- Use comments to leave notes to yourself
- Mark sections that need review
- Track research sources

### For Reviewers

- Be specific about what needs changing
- Reference line numbers or text
- Resolve comments when addressed

### For Collaboration

- Set distinct profile colors
- Use clear, actionable language
- Resolve comments when complete

## Exporting with Comments

When you export:

- **Markdown:** Comments are stripped
