Keyboard Shortcuts

Keyboard Shortcuts

Master Korppi with these keyboard shortcuts.

File Operations

Shortcut Action
Ctrl+N New document
Ctrl+O Open document
Ctrl+S Save document
Ctrl+Shift+S Save As
Ctrl+W Close current tab

Editing

Shortcut Action
Ctrl+Z Undo
Ctrl+Y Redo
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste
Ctrl+A Select all

Text Formatting

Shortcut Action
Ctrl+B Bold
Ctrl+I Italic
Ctrl+U Underline (if supported)

Find & Replace

Shortcut Action
Ctrl+F Open Find panel
Ctrl+H Open Find & Replace
F3 Find next match
Shift+F3 Find previous match
Ctrl+G Go to next match
Ctrl+Shift+G Go to previous match
Escape Close Find panel

In Find Panel

Shortcut Action
Enter Find next
Shift+Enter Find previous

Navigation

Shortcut Action
Ctrl+Home Go to document start
Ctrl+End Go to document end
Ctrl+Tab Next document tab
Ctrl+Shift+Tab Previous document tab

View

Shortcut Action
Ctrl+Plus Zoom in (if supported)
Ctrl+Minus Zoom out (if supported)
Ctrl+0 Reset zoom

Special Actions

Shortcut Action
Ctrl+Right-Click Show browser debug menu

Context Menu Shortcuts

When text is selected:

  • Right-click → Formatting options
  • Bold, Italic, Strikethrough, etc.
  • Add Link, Add Comment, Copy

When no text is selected:

  • Right-click → Insert options
  • Paste, Table, Image, Code Block

Tips

Learning Shortcuts

  1. Hover over toolbar buttons to see shortcuts
  2. The shortcut appears in the tooltip
  3. Practice common ones first

Most Used

Focus on these essential shortcuts:

  1. Ctrl+S - Save (muscle memory!)
  2. Ctrl+Z / Ctrl+Y - Undo/Redo
  3. Ctrl+B / Ctrl+I - Bold/Italic
  4. Ctrl+F - Find

macOS Users

Replace Ctrl with Cmd (⌘):

  • Cmd+S - Save
  • Cmd+B - Bold
  • etc.

Customization

Custom keyboard shortcuts are planned for a future release.
